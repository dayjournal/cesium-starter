# cesium-starter

![README02](img/README02.png)

Start Cesium easily.  
- [Cesium v1.107.0](https://github.com/CesiumGS/cesium)  
- [TypeScript v5.1.6](https://www.typescriptlang.org)  
- [Vite v4.4.1](https://vitejs.dev)  
- node v18.1.0
- npm v8.19.2

<br>

## Usage

![README03](img/README03.png)

<br>

Install package
```bash
npm install
```

<br>

build
```bash
npm run build
```

<br>

dev
```bash
npm run dev
```
