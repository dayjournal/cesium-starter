# cesium-starter

![README02](img/README02.png)

Start Cesium easily.  
- [Cesium v1.97.0](https://github.com/CesiumGS/cesium)  
- [TypeScript v4.8.2](https://www.typescriptlang.org)  
- [Vite v3.1.0](https://vitejs.dev)  
- node v18.1.0
- npm v8.8.0

<br>

## Usage

![README03](img/README03.png)

<br>

Install package
```bash
npm install
```

<br>

build
```bash
npm run build
```

<br>

dev
```bash
npm run dev
```
